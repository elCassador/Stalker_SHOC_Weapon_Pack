## Requirements

For the mod to work correctly, **MultipleHUD v1.04** **must** be installed and placed in the `Mods` folder.

---

## Controls

* Load ammo into magazine: **Hold LMB**
* Unload ammo from magazine: **Hold RMB**
* Insert magazine into rifle: Put magazine into **Utils** and press **R**
* Remove magazine from rifle: Make sure hotbar has free slot and press **R**
